To perform a release of the Codeprimate Elements (cp-elements) project, run the following script:

0. Set the project's 'Major, Minor, Maintenance and Version Qualifier' for the release (e.g. 1.0.0.RELEASE)
in both pom.xml and gradle.properties.

Version Qualifier maybe 1 of: M# (Milestone #; e.g. M2), RC# (Release Candidate #; e.g. RC1) or RELEASE.

1. Build with all supported JDK [LTS] versions from baseline forward, for example: [JDK 8, JDK 11, JDK14, JDK17, JDK18].

2. `mvn -P code-analysis,prepare-release clean install site`

3. `gradlew [--no-daemon] clean build`

4. Copy 'cp-elements/target/site' to 'cp-core-workspace/etc/releases/cp-elements-<major>.<minor>.<maint>.<version-qualifier>/'.

5. Copy 'cp-elements/build/reports' to 'cp-core-workspace/etc/releases/cp-elements-<major>.<minor>.<maint>.<version-qualifier>/'.

6. Copy 'cp-elements/target/apidocs' to 'cp-core-workspace/etc/releases/cp-elements-<major>.<minor>.<maint>.<version-qualifier>/'.

7. Copy 'changelog.txt' to 'cp-core-workspace/etc/releases/cp-elements-<major>.<minor>.<maint>.<version-qualifier>/'.

8. [Optional] Copy Checkstyle, PMD and SpotsBugs configuration files and results to 'cp-core-workspace/etc/releases/cp-elements-<major>.<minor>.<maint>.<version-qualifier>/'.

9. `mvn -P release deploy [-DskipTests]`

10. Login to Sonatype Nexus Repository Server and "release" artifacts (contained in the staging repository) to Maven Central.

11. Send announcement.
